# Locker

Minecraft Sponge Plugin for locking chests and other blocks.

# Supported Blocks:

    - CHEST
    - ENDER_CHEST
    - TRAPPED_CHEST 
    - FENCE_GATE
    - ACACIA_FENCE_GATE
    - BIRCH_FENCE_GATE
    - JUNGLE_FENCE_GATE
    - SPRUCE_FENCE_GATE
    - DARK_OAK_FENCE_GATE
    - WOODEN_DOOR
    - DARK_OAK_DOOR
    - ACACIA_DOOR
    - BIRCH_DOOR
    - JUNGLE_DOOR
    - SPRUCE_DOOR
